# 👥 통합 UX 개선 - 업주도 소비자 기능 사용

> **목표**: 업주도 소비자처럼 앱을 사용하고, 마이페이지에서 "사장님 페이지" 접근
> **난이도**: ⭐⭐ 쉬움
> **예상 시간**: 10분

---

## 📋 변경 사항

### Before (기존):
```
업주 로그인 → 업주 대시보드만 표시 (소비자 기능 사용 불가)
소비자 로그인 → 소비자 화면만 표시
```

### After (개선):
```
업주 로그인 → 소비자 화면 표시 (업체 탐색, 예약 가능)
           → 마이페이지 → "사장님 페이지" 버튼 → 업주 기능 접근

소비자 로그인 → 소비자 화면 표시 (기존과 동일)
```

---

## 🎯 주요 기능

### 1️⃣ 마이페이지 추가
- **프로필 정보** 표시
- **예약 내역** 바로가기
- **사장님 페이지** 버튼 (업주만)
- **로그아웃** 기능

### 2️⃣ 사장님 페이지
- 마이페이지에서만 접근 가능
- 상품 관리, 예약 관리, 캐시 관리
- 뒤로가기 → 다시 마이페이지로

### 3️⃣ HomeScreen 개선
- 우측 상단 "👤" 아이콘 → 마이페이지
- 하단 "내정보" 메뉴 추가

## 🎯 테스트 방법

### 1️⃣ 업주 계정으로 테스트

1. **로그인:**
   - 이메일: storeowner@gmail.com
   - 비밀번호: store1234

2. **소비자 기능 사용:**
   - ✅ HomeScreen 표시됨
   - ✅ "업체 보기" 클릭 → 다른 업체 탐색 가능
   - ✅ 예약 가능

3. **마이페이지 접근:**
   - 우측 상단 "👤" 아이콘 클릭
   - 또는 하단 "내정보" 메뉴 클릭

4. **사장님 페이지 접근:**
   - 마이페이지에서 "🏪 사장님 페이지" 버튼 클릭
   - ✅ 업주 대시보드 표시
   - 상품 관리, 예약 관리, 캐시 관리 가능

5. **돌아가기:**
   - 대시보드 "로그아웃" 버튼 클릭
   - → 다시 마이페이지로 이동

---

### 2️⃣ 소비자 계정으로 테스트

1. **로그인:**
   - 일반 소비자 계정

2. **마이페이지 접근:**
   - 우측 상단 "👤" 아이콘 클릭

3. **확인:**
   - ✅ "사장님 페이지" 버튼 **없음** (소비자는 안 보임)
   - ✅ 예약 내역, 로그아웃 등만 표시

## 📸 화면 구조

### 전체 흐름:

```
로그인
  ↓
HomeScreen (모든 사용자 공통)
  ├─ 업체 보기
  ├─ 예약 내역
  └─ 내정보 (👤)
      ↓
    MyPageScreen
      ├─ 예약 내역
      ├─ 🏪 사장님 페이지 (업주만)
      │    ↓
      │  StoreDashboard
      │    ├─ 상품 관리
      │    ├─ 예약 관리
      │    └─ 캐시 관리
      └─ 로그아웃
```

## 🎨 UI 디자인

### MyPageScreen:
- **프로필 섹션**: 둥근 아이콘, 이름, 이메일
- **사장님 페이지 버튼**: 주황색 강조 (업주만)
- **절약 금액**: 초록색 카드 (소비자)
- **메뉴 리스트**: 아이콘 + 텍스트
- **로그아웃**: 빨간색 버튼

### HomeScreen:
- **헤더**: 파란색 배경
- **마이페이지 아이콘**: 우측 상단 둥근 버튼
- **메인 버튼**: 큰 카드 레이아웃
- **하단 메뉴**: 내정보 바로가기
