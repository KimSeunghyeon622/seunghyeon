### 시나리오 1: 전체 플로우 (예약 → 픽업 → 리뷰 → 답글)

#### 1단계: 소비자 예약
1. **로그인**: `consumer@test.com` / `test1234`
2. **업체 선택**: "프레시 밀키트" 또는 "테스트베이커리"
3. **제품 예약**: 수량 선택 → 픽업 시간 선택 → 예약하기
4. **확인**: "나의 예약" 탭에서 예약 확인

#### 2단계: 업주 픽업 완료 처리
1. **로그인**: `owner3@test.com` (프레시 밀키트) 또는 `storeowner@gmail.com`
2. **예약 관리**: "내 매장 예약" 탭 선택
3. **픽업 완료**: 예약 확인 → "픽업 완료 처리" 버튼 클릭

#### 3단계: 소비자 리뷰 작성
1. **로그인**: `consumer@test.com` / `test1234`

3. **리뷰 작성**: 별점 + 내용 입력 → 제출

#### 4단계: 업주 답글 작성
1. **로그인**: `owner3@test.com` / `test1234`
2. **리뷰 관리**: 새 리뷰 확인
3. **답글 작성**: 답글 입력 → 제출
4. **확인**: 업체 상세 페이지에서 답글 확인
