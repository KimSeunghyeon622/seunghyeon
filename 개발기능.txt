[웹앱]

  Expo Web 기반으로 PWA를 추가하면 앱/웹 동시 지원이 가능합니다.
  현재 앱 코드 재사용률이 높고, “앱 형태로 보이는 웹”을 만들 수 있습니다.

  ———

  ## 우리가 해야 할 일 (요약 로드맵)

  ### 1) Expo Web 활성화

  - 기존 Expo 프로젝트에서 expo start --web 가능하게 설정
  - 라우팅/네비게이션은 현재 구조 그대로 유지 (App.tsx 기반)

  ### 2) 웹 전용 UI 보정

  - 모바일 UI를 그대로 웹에 보여주면 깨지는 부분을 수정
  - 클릭 영역, 스크롤, 폰트, 레이아웃을 웹 최적화

  ### 3) PWA 설정

  - 홈화면 아이콘, 앱 이름, 스플래시, offline 기본 처리
  - “웹앱처럼” 주소창 최소화, 전체화면 모드

  ### 4) 기능 동일성 확보

  - 로그인/예약/리뷰/결제/푸시 알림 등 핵심 기능 확인
  - 웹에서 동작 안 되는 기능은 대체 UX 제공

  ———

  ## 추가로 확인할 것

  TRD 기준으로 보면 웹앱은 아직 범위에 없어서, 아래를 정리해야 합니다:

  1. 웹에서 사용할 도메인
  2. 푸시 알림 웹 대응 여부 (모바일만 유지할지)
  3. 결제(토스 등) 웹 지원 필요 여부

  ———
